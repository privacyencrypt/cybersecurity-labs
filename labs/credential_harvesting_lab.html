Overview

This lab aligns with the CompTIA Security+ exam objectives, particularly:

  • Domain 1.1: Common types of attacks (phishing, credential harvesting).
  • Domain 1.5: Social engineering techniques and mitigation strategies.

We will use tools like SEToolkit to simulate credential harvesting and explore defensive mechanisms to identify and prevent these attacks.

Key Objectives

  • Understand the process of credential harvesting using SEToolkit.
  • Learn how to configure site cloning for phishing simulations.
  • Understand detection and mitigation strategies aligned with Security+ objectives.

Tools Used

  • Kali Linux: A penetration testing operating system.
  • SEToolkit: A tool for creating phishing simulations.
  • Network Configuration Tools: For setup and IP management (e.g., ifconfig).

Step-by-Step Instructions

Open Kali Linux in a virtual machine and ensure networking is configured properly (NAT or Bridged mode). Run the following command to update your tools:

sudo apt update && sudo apt upgrade -y

Why: Ensures the latest security patches are applied.

Start SEToolkit with the following command:

sudo setoolkit

If prompted, enter your root password. From the menu, select 1: Social-Engineering Attacks.

Select the following options:

  1. 2: Website Attack Vectors
  2. 3: Credential Harvester Attack Method
  3. 2: Site Cloner

Enter your IP address using ifconfig and the target URL, e.g., https://example.com.
