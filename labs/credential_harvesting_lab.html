Overview

This lab focuses on credential harvesting, a critical attack vector in cybersecurity. It aligns with:

  • Security+ Domain 1.1: Threats, attacks, and vulnerabilities.
  • Security+ Domain 1.5: Social engineering and mitigation techniques.

We simulate phishing attacks using SEToolkit and discuss their detection and prevention.

Key Objectives

  • Understand the mechanics of credential harvesting through phishing.
  • Gain hands-on experience with SEToolkit.
  • Learn to identify and mitigate credential-harvesting attacks.

Tools Used

  • Kali Linux: Penetration testing OS.
  • SEToolkit: Social engineering tool.
  • Network tools: For IP management and configuration.

Step-by-Step Instructions

Open Kali Linux in a virtual machine. Ensure networking is set to NAT or bridged mode. Update tools:

sudo apt update && sudo apt upgrade -y

Start SEToolkit:

sudo setoolkit

Select 1: Social-Engineering Attacks from the menu.

Select options in order:

  1. 2: Website Attack Vectors
  2. 3: Credential Harvester Attack Method
  3. 2: Site Cloner

Enter your IP address (ifconfig) and the target URL (e.g., https://example.com).

Send the cloned site link to a test user. Captured credentials will appear in the SEToolkit terminal.
